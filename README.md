# Aprendizado - Java

Este repositório serve como um histórico da minha evolução na programação. Aqui guardo classes, funções e algoritmos que desenvolvo enquanto estudo os fundamentos da linguagem.

## O que tem aqui?

- **Lógica de Programação:** Exercícios de repetição, condicionais e operadores.
- **Estruturas de Dados:** Implementações de listas, arrays e manipulação de coleções.
- **Utilidades:** Funções que podem ser reaproveitadas em projetos futuros.

## Tecnologias

- Java
- Git & GitHub
